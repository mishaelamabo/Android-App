# Android-App
Student Grading Calculator
A comprehensive desktop application built with Kotlin and Swing GUI for managing student grades across four core courses.The system allows educators to register students with their name, matricule number, and email, then record CA marks (0-30) and exam marks (0-70) for each course. Grades are automatically calculated based on a detailed scale from A (4.0) for scores 80-100 down to F (0.0) for scores 0-39, with intermediate grades including B+, B, C+, C, D+, and D. The application features a tabbed interface with sortable data tables for viewing all records and detailed individual student transcripts showing course-by-course breakdowns. Real time GPA calculation provides both individual student averages and overall class statistics displayed in a summary panel. Built with clean architecture separating models, UI, and business logic, the system includes comprehensive input validation to prevent invalid entries, duplicate matricule numbers, and out of range marks. The intuitive interface, real time updates, and automated calculations make it an efficient tool for educational institutions to track and manage student performance.
